Tableau 중급 대시보드 액션2탄 소셜미디어

* 엑셀파일 가져와서 연동시킬 수 있음

1)
[소셜 미디어채널] Shape 마크에 끌어옴
[소셜 미디어채널] 칼럼으로 이동 - 전체 보기 
Shape 마크 - Select Shape Palette - 소셜 미디어 채널 - 클릭으로 1:1 매칭 
상단 필드 레이블 숨기고 - 항목 글꼴 키우고 - Format - 테두리 - Row Divider - None 테두리 없애기
시트명 -> SNS 
2)
새시트에 [누적 구독자 수] 더블클릭 - [월간] 마우스 오른쪽 누르고 칼럼으로 연속형Month 선택
[소셜 미디어 채널] 색상 마크로 끌기 - SNS 아이콘 색상과 동일하게 변경 - [누적 구독자수] 레이블로 끌기
Format - 라인 - Grid Lines - None 
Sheet - Axis Rulers - 굵은 검정
시트명 -> SNS 월간 구독자수

3)
대시보드 만들기
상단 SNS 하단 그래프
필터들 지우기
Dashboard - Actions - Add Actions - Highlighting
소셜 미디어 채널 하이라이팅 - SNS체크 - Hover - Target Sheet(구독자수) / 아이콘에 마우스오버하면 라인그래프가 표시

시트에서 [URL] Detail 마크로 끌고오기 
Dashboard - Actions - Add Actions - Go to URL 
해당 소셜 미디어 채널로 이동하시겠습니까? - SNS 체크 - Menu - URL(URL) 

-----------------------------------------------------------
하이라이트 + URL로 이동
1. 데이터 연결
   누적 고객수 시트 정리 후 소셜미디어 채널 URL 시트와 조인
2. 소셜 미디어 채널 모양 워크 시트 만들기
3. 월간 누적 구독자 수 워크 시트 만들기
4. 새 대시보드 만들어 두 워크 시트 올리기 
5. 메뉴의 대시보드 작업에서 하이라이트 추가
6. 메뉴의 대시보드 작업에서 대시보드 작업 추가
 Tip) 채널 모양 워크 시트 세부 정보 마크에 URL 정보 포함하기
